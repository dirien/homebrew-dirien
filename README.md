# homebrew-dirien

A homebrew tap for MacOS users for different clis

## Usage

Install the following apps via this Homebrew tap:

* k3sair 🏴‍☠️
* minectl 🗺


### Install tap

```bash
brew tap dirien/homebrew-dirien
````

### Install k3sair 🏴‍☠️

```bash
brew install k3sair
```

### Install minectl 🗺

```bash
brew install minectl
```
